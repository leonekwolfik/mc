mc — Midnight Commander

The classic two-panel file manager, running in your browser via a lightweight Docker container.

View on GitHub Docker Hub
🖥️

Browser-based UI

Access Midnight Commander from any device — no SSH client or local installation needed.

🐳

Docker-native

One command to run. Works with Docker CLI, Docker Compose, or any container platform.

🗂️

Volume support

Mount your local directories directly into the container and browse them with MC.

🪶

Alpine & Ubuntu

Choose between a minimal Alpine image or a full Ubuntu base, depending on your needs.

Quick Start

Pull the image

docker pull leonekwolfik/midnight-commander

Run the container

docker run -d \
  --name midnight-commander \
  -p 7681:7681 \
  leonekwolfik/midnight-commander

Open in your browser

Navigate to http://localhost:7681

Usage

Mount a local directory

The container auto-detects the first mounted volume and opens MC there.

docker run -d \
  --name midnight-commander \
  -p 7681:7681 \
  -v /path/to/your/files:/data \
  leonekwolfik/midnight-commander

docker compose

Clone the repo, then:

docker compose up -d

The included docker-compose.yml mounts a local data/ folder and maps port 7681.

Build from source

# Alpine (recommended — smaller image)
docker build -f Dockerfile.alpine -t leonekwolfik/midnight-commander .

# Ubuntu
docker build -f Dockerfile.ubuntu -t leonekwolfik/midnight-commander .
